Case Study: Coasian Bargain, Indonesia

The data below is a JSON object that consists of 4 key-value pairs:

  • "unilateral_cost", which contains a float value of the unilateral cost of the country doing the unilateral action.
  • "unilateral_benefit", which contains a float value of the unilateral benefit of the country doing the unilateral action.
  • "freeloader_benefit", which contains a JSON object with the countries doing the unilateral action as the keys, and the values being the JSON objects of 
    {"<'freeloader' country name>": <benefit amount>}
  • "global_benefit", which contains a float value of the fraction of global benefit, scaled by a fraction of the country's social cost of carbon, as the values.
The values are in billion dollars.
